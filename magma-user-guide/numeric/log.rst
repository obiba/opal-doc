log
===

Returns the base 10 (or a specific base) logarithm of the value.

See also :doc:`ln`.

Syntax
------

.. code-block:: javascript

  log([base])

=============== ============================
Parameter       Description
=============== ============================
``base``        The optional base of the logarithm to evaluate (default is ``10``).
=============== ============================

Examples
--------

.. code-block:: javascript

  $('AVar').log()
  $('AVar').log(2) // base-2 logarithm
