pow
===

Returns the value raised to the power of the argument.

Syntax
------

.. code-block:: javascript

  pow([power])

=============== ============================
Parameter       Description
=============== ============================
``power``       The power to raise the value to.
=============== ============================

Examples
--------

.. code-block:: javascript

  $('AVar').pow(2) // AVar * AVar
  $('AVar').pow(-1) // 1 / AVar
