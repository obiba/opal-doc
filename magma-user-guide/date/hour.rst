hour
====

Returns the hour of the day for the 12-hour clock (0 - 11). Noon and midnight are represented by 0, not by 12. For example, at 10:04:15.250 PM the HOUR is 10.

See also :doc:`hourOfDay`.

Syntax
------

.. code-block:: javascript

  hour()

Examples
--------

.. code-block:: javascript

  $('Date').hour()
