add
===

Adds days to a value of date or date time type.

Syntax
------

.. code-block:: javascript

  add(days)

=============== ============================
Parameter       Description
=============== ============================
``days``        The number of days to be added.
=============== ============================

Examples
--------

Adds 2 days:

.. code-block:: javascript

  $('Date').add(2)

Subtracts 4 days:

.. code-block:: javascript

  $('Date').add(-4)
